# ChatGPT被降智？用户懵了，chatgpt到底搞了啥幺蛾子？

最近，关于ChatGPT降智的话题闹得沸沸扬扬。所谓“降智”，简单来说就是：ChatGPT变笨了！ 它不但回答问题显得更“懒”，甚至一些高级功能也直接“罢工”。

比如联网搜索、图片解析、文生图等。更让人无语的是，这事儿就算你开了 ChatGPT plus会员也照样没用。该不回答还是不回答，主打一个高冷！

ChatGPT降智到底是怎么回事？咱们今天好好唠一唠。

# 什么是ChatGPT降智？

ChatGPT降智就是让模型智商直线下降，用起来处处不顺。比如：

  1. 没有“思考”时间：用o1模型推理时，本来应该显示“思考了xx秒”，结果现在直接蹦出答案。连“思考”的样子都懒得装了，妥妥底层模型被偷换。

  2. 图片解析跪了：上传图片，ChatGPT一句“我看不懂”把你打发。

  3. 文件解析拉跨：文件发上去，得到的却是冷冰冰的“无法处理”。

  4. 文生图不见了：让它画张图，直接拒绝。这可是Pro套餐的“标配功能”啊！

这些表现，完全不像“高智商选手”该有的操作。

# ChatGPT降智的幕后原因

说白了，ChatGPT的“降智”本质上是资源问题。最新数据显示，ChatGPT的周活跃用户已经突破了2亿，这么多用户在线，chatgpt的算力压力可想而知。

再加上它还得花资源训练新模型、维持业务，然而公司本身却连年亏损，预计2024年亏损额可能高达50亿美元！

在这样的背景下，chatgpt只能选择降智部分用户的服务，偷偷摸摸给你换了个低配版的模型，同时砍掉一些高级功能——节约算力，从你我这种“普通用户”身上省点力。

# 观察到的ChatGPT降智手段

目前来看，ChatGPT降智的主要操作手法有以下两种：

  1. 偷换底层模型：最典型的例子，就是把本来的o1模型换成了更低算力的GPT-4o，甚至再降级到GPT-4o mini或GPT-3.5 mini。表面看回答还在，但你会觉得它的表现“怪怪的”，回答质量明显下降。

  2. 限制高级功能：联网搜索、图片解析、文生图这些高级功能一刀砍掉。如果你感觉最近用ChatGPT时总是功能缺失，十有八九是被“精准降智”了。

# 受ChatGPT降智影响的用户

目前来看，以下几类用户最容易“中招”：

  1. 中国用户：这个群体反馈最为集中，英文社区几乎没什么类似讨论。
  2. 网页端用户：这波ChatGPT降智主要体现在ChatGPT的网页端，手机App暂时未受影响。
  3. IP多用户：一些IP使用人数较多（比如公司网络）的用户更容易被盯上。
  4. 账号维度限制：换IP可能有用，但如果账号被标记，那基本无解。

# 如何应对ChatGPT的降智？

针对ChatGPT降智，目前有一些解决方案可以一试：

  1. 转用手机App或Mac客户端：如果你习惯用网页端，可以试试转战手机App或桌面客户端。目前来看，这两个平台的功能都正常没有发现拒绝回答或者答非所问的情况。

  2. 更换IP：切换一个更干净的IP可能有帮助，但无法保证100%有效，毕竟chatgpt可能已经对账号本身进行了标记。

  3. 尝试上传图片：之前有网友发现，给ChatGPT上传一张图片，有时能“唤醒”GPT-4o的功能，暂时恢复正常。不过，这个方法最近的成功率大幅下降，仅供参考，如果起作用，纯属碰巧。

  4. 战术性放弃ChatGPT：如果实在用得憋屈，完全可以转投其他家的服务，比如Claude、Gemini、xAI等。特别推荐Claude，它不仅功能强大，还有一些独特的亮点，比如Artifacts和Project功能，谁用谁知道。Claude如何开通，如何订阅使用可以看看这篇文章：《Claude是什么？怎样注册Claude账号？Claude Pro怎么订阅支付？》 (opens new window)

# 写在最后

ChatGPT降智的锅，大概率要归结于chatgpt的资源压力和盈利困境。虽然用户体验受到了不小的影响，但也算是无奈之举。比较靠谱的做法是切换IP，或者重新申请账号使用。
