# ClashX Pro 是什么？Dler Cloud 怎么订阅？最全使用教程来了！

# 先聊聊 ClashX Pro

大家都知道，网络环境越来越复杂，很多网站或者内容你在国内可能没法正常访问。ClashX Pro 就是一款专门为此设计的工具。它的主要功能是帮助你通过代理协议科学上网。什么是代理协议？简单说，就是借助第三方服务器把你的网络请求绕个道，从而实现访问受限内容。

ClashX Pro 是基于开源项目 Clash 开发的，支持各种代理协议，比如 Shadowsocks、V2Ray、Trojan 等。对于 Mac 用户来说，它已经是不可或缺的工具了，设置简单、功能强大，支持 macOS、Windows、Linux、安卓甚至 iOS 系统。

# Dler Cloud 又是什么？

说到 ClashX Pro，不得不提它的好搭档——Dler Cloud (opens new window)。它是一个老牌的网络加速服务（也叫“机场”），成立于 2018 年，专门为用户提供高速稳定的网络代理服务。

Dler Cloud 支持多种代理协议，像 Shadowsocks（SS）、ShadowsocksR（SSR）、V2Ray、Trojan 等，线路也特别丰富，有普通的直连节点、专线节点，甚至还有为企业设计的动态专线。所以，不管你是用来看 Netflix、Disney+ 还是 YouTube Premium，它都能轻松解锁。

最贴心的是，Dler Cloud 提供按量付费的套餐，按你实际用的流量收费，用多少算多少，特别适合偶尔才用的人。

# 怎么订阅 Dler Cloud？

说了这么多，怎么才能订阅 Dler Cloud 呢？其实特别简单，几分钟就能搞定：

  1. 登录官网：Dler Cloud 官方网站 (opens new window)

  1. 充值：选择支付宝充值，充值金额最好大于￥498。这样可以购买年套餐，性价比高。

  1. 选择套餐：充值成功后，选择适合你的套餐。比如，作者选的是￥498/年套餐，这个套餐适合国内使用，不然选择的太低可能会有使用限制。

  1. 开始使用：购买成功后，就能在 ClashX Pro 上添加订阅地址，轻松上网了。

# 如何配置 ClashX Pro？

订阅完 Dler Cloud 后，接下来就是配置 ClashX Pro。别担心，步骤很简单：

  1. 登录 ClashX Pro：打开软件，用你的 Dler Cloud 账号（邮箱）和密码登录，软件会自动加载配置。

  1. 手动设置代理：为了安全和稳定，建议你手动选择代理，而不是使用自动测试（auto-urltest）功能。

  1. 系统代理设置：在 ClashX Pro 设置中，勾选“设置为系统代理”，这样你整个系统的网络流量都会经过代理，确保每个应用程序都能使用。

  1. 更新配置：如果使用中遇到问题，或者发现连接速度变慢，可以去托管配置页面，点击刷新配置来更新。

# 账号共享可以吗？

这里有个大家比较关心的问题，Dler Cloud 账号能共享吗？ 答案是账号本身不能共享，但配置文件是可以的。你可以把配置文件分享给朋友，他们用这个配置文件就能像你一样使用 Dler Cloud 的服务。不过一个账号不能在多个设备上同时登录。
