# 申请chatgpt、ChatGPT账号

首先打开 chatgpt 的账户注册页面 (opens new window)选择signup输入邮箱。

输入邮箱密码后，成功发送邮件验证，直接点击页面「Open Gmail」，跳转到谷歌邮件页面，找到chatgpt发送的验证邮件，点击进行邮箱验证即可。

补充完善个人信息，点击Agree，chatgpt账号就注册好了。

然后选择您创建 OpenAPI 还是 ChatGPT账户，一般不是研究人员都选择创建ChatGPT账户。
